# Changelog

Changes from v13.8.7 to v13.8.8:

## Features

- **Artist blocking** — block or unblock artists from the artist screen; blocked content is hidden across home and explore
- **App icon picker** — choose your launcher icon under Settings → Appearance → App icon (community icons supported when added to the catalog)
- **Home improvements** — tonal backdrop option, cleaner loading/empty/error states, and smoother category browsing

## Playback & lyrics

- Skip unreliable external audio fallback when YouTube bot detection is triggered
- Provider lyric translations only show for AI-translated lyrics (less clutter on synced lyrics)

## Search & navigation

- Tap the Search tab again while already on Search to open the search bar instantly

## Settings

- Updated synced lyrics library (accompanist lyrics-core 0.4.7)

## Upstream credits
