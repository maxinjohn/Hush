# Changelog

Changes since v13.11.7:

## Highlights

- **Waze auto-pause fix** — removed unnecessary `player.prepare()` calls after seek/next/previous Waze commands that caused brief audio dropouts when switching songs or shim apps. *(Hush)*
- **Faster downloads** — adapted 5MB chunked download data source from Echo Music to bypass YouTube per-request throttling. *(Hush), ported from Echo Music*
- **Offline playback optimisation** — `isOfflinePlayback` flag skips unnecessary `playerResponseForMetadata` and `YouTube.next()` API calls when playing from cache or completed downloads. *(Hush), ported from Echo Music*
- **Wider navigation bar** — bottom toolbar now fills screen width edge-to-edge instead of being capped at 520dp with 16dp horizontal padding. *(Hush)*
- **Playlist sync deduplication** — remote playlists are now deduplicated by ID during sync to prevent duplicate entries. *(Hush), ported from Metrolist*

## Waze Bridge

- **Auto-pause fix** — `WazeCommandReceiver` no longer calls `player.prepare()` after `seekToNext()`, `seekToPrevious()`, and `seekTo()`; `MusicService.playWazeQueueItem()` also removed the redundant `prepare()` call. *(Hush)*

## Downloads & Playback

- **5MB chunked downloads** — new `ChunkingDataSource.kt` wraps `OkHttpDataSource` to split HTTP requests into 5MB range chunks, dramatically improving download speeds by bypassing YouTube throttling. *(Hush), ported from Echo Music*
- **Offline playback flag** — `recoverSong()` now accepts `isOfflinePlayback: Boolean = false`. When playing from cache (`resolveLocalPlaybackDataSpecIfAvailable`, `cachedPlaybackUrl`, `cached extractor URL`), it skips `playerResponseForMetadata` and related-song discovery. *(Hush), ported from Echo Music*

## Sync

- **Playlist deduplication** — `syncSavedPlaylists()` now calls `.distinctBy { it.id }` on remote playlists to prevent duplicate entries during sync. *(Hush), ported from Metrolist*

## UI

- **Wider navigation bar** — removed 520dp max width constraint and 16dp horizontal padding from `FloatingNavigationToolbar` for full-width edge-to-edge navigation bar. *(Hush)*

## Settings

- **Data Saver preference key** — `DataSaverEnabledKey` boolean preference added, ready for full Data Saver mode integration. *(Hush)*

## Housekeeping

- Version bumped to 13.11.8 (versionCode 158). *(Hush)*
- Waze Bridge shim bumped to versionCode 160. *(Hush)*
