# Changelog

Changes in v13.8.4..v13.8.5:

## Fixes

- **Regional lyrics:** Malayalam and other Indian-language lyrics display correctly again; only clearly wrong languages (e.g. Arabic or Japanese on Indian tracks) are filtered out
- **Like/unlike while playing:** Instant local heart toggle; syncs to YouTube Liked Music in the background without blocking playback
- **Notification & lock screen artwork:** Album art is set immediately and embedded for Android system media controls
- **Release notes generation:** Fixed CI changelog script that produced empty bullet sections

## UI

- Removed redundant **Start radio** from playlist, liked, and song long-press menus; unified actions around **Play** and **Shuffle**
- Square, modern menu action buttons with haptic feedback
- Larger mini player (64dp bar, bigger artwork)
- Slightly larger home quick-picks carousel

## Performance

- Faster local/CI builds via improved Gradle worker settings

## Other

- Centralized YouTube like sync through `SyncUtils` (no duplicate API calls)
- Added categorized release notes generator for future releases
