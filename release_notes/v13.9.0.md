# Changelog

Changes in v13.8.9..v13.9.0:

## Features

- **Audio Visualizer** — 7 real-time bar styles (Bottom Bars, Color Wave, Glow Bars, Dual Bars, Pulse Bars, Spectrum, Jumping Bars) powered by Android Visualizer FFT. 5 color themes, MiniPlayer support. *(Hush)*
- **Custom Parametric Equalizer** — 10-band peaking/low-shelf/high-shelf EQ with biquad filters, real-time frequency response curve, preset manager, AutoEq import, bass boost, virtualizer, CSV import/export. *(Hush)*
- **SponsorBlock Integration** — auto-skip or manual skip button for sponsors, intros, outros, and 11 categories. *(Hush)*
- **Ocean Wave Lyrics** — Apple Music-style animated lyrics with continuous wave sweep, trailing gradient, progressive blur. *(Vivi Music)*
- **JioSaavn Streaming Overhaul** — 3-path resolution (auth token → DES v4 → DES flat), multi-format search with 4 response parsers, automatic server retry. *(Hush)*
- **JioSaavn Device Router** — device-aware API server selection with remote config and automatic fallback. *(Hush)*
- **CDN Failover** — automatic YouTube CDN hostname rotation on 403-530 errors. *(Hush)*
- **Remote Cipher Config** — centrally-managed YouTube cipher overrides and disabled client sets. *(Hush)*
- **Home Screen Chip Loading** — instant chip selection feedback, shimmer placeholders, snackbar error notifications. *(Hush)*

## Player

- **Visualizer overlay** — spectrum bars rendered over album art in full player and mini-player. *(Hush)*
- **Shuffle/repeat row** — added to landscape player controls. *(Hush)*
- **Download button** — in landscape secondary actions for non-local media. *(Hush)*

## Settings

- Visualizer controls added to Appearance (enable, style, color theme, opacity, mini-player). *(Hush)*
- JioSaavn moved from Debug to Player settings with dedicated navigation route. *(Hush)*
- Nerd Stats section in Debug (playback source indicator, audio format). *(Hush)*
- V7/V8/V9 design styles deprecated. *(Hush)*

## Fixes

- **PreferenceGroup** — defensive copy prevents ConcurrentModificationException on DataStore updates. *(Hush)*
- **SaavnSearch** — no longer filters out songs without immediately available download URLs. *(Hush)*
- **SaavnUrlDecryptor** — deduplication and additional URL quality fallback patterns. *(Hush)*
- **JioSaavn Settings** — UI cleanup with simplified layout and compact spacing. *(Hush)*

## Housekeeping

- All font files subsetted (avg 70% size reduction); font families default to system font. *(Hush)*
- Kuromoji IPA dictionary removed; Japanese romanization uses Katakana→Romaji map. *(Hush)*
- Poppins typeface removed from Markdown rendering. *(Hush)*
- SansFlex and GoogleSans font options removed from FontSelection. *(Hush)*
- 56 new string resources for EQ, visualizer, and lyrics settings. *(Hush)*
