# Contributing to Hush

Hush is a **personal fork** maintained for private use on my own devices.

Please contribute to upstream instead:

- **Code & bug reports:** https://github.com/ArchiveTuneApp/ArchiveTune
- **Donations:** upstream ArchiveTune maintainers

You may open an issue here if a Hush release build is broken for you. For broader fixes and features, use upstream ArchiveTune — I periodically merge changes back into Hush.
